要求实现一个超市收银系统
星期一三正常收费
星期二四全场9折
星期五 满500减100
星期六 非生鲜类商品全部9折,全场商品满300减50
星期天 生鲜类商品8折且满500减200 非生鲜类商品打9折,满300减50
  先打折  再满减
  // 先计算商品总值,再打折,再满减


需求分析:
1.商品需要放到购物车中在收银台进行统一收费
2.商品需要放到购物车中,商品分为生鲜类和非生鲜类
3.商品结算时需根据当日的收费标准进行计算
  折扣分为两种   满减 和 折扣

UML建模:
  1.购物车类: 用于存放商品
  2.抽象商品类  暂时有生鲜类和非生鲜类
  3.抽象商品价格计算类
